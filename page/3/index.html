TensorFlow-import-data

Posted on 2017-10-10 | Edited on 2019-05-03 | In Tensorflow

Low level APIs

学会使用低层次的tensorflow编写程序，包含以下几个方面的内容：

  • 自己管理tensorflow程序(tf.Graph)并运行自己的tensorflow程序(tf.session)，而不是使用TensorFlow中默认的估计器(Estimator)去管理他们
  • 使用tf.session运行TensorFlow operation
  • 在低层次环境中使用高层次组件：datasets,layers,features_columns
  • 构建自己的训练循环，而不是使用预制的估计器(Estimator)

首先我们推荐使用高层次的API来构建自己的模型，即使用预制的Estimator来管理和运行模型， 但是了解TensorFlow的内核有以下的几点好处：

  • 当直接使用低层次的TensorFlow APIs构建模型时，验证和调试起来会对模型的性能和评价有更直观的理解
  • 当只用高层次API构建模型时，你会更加知道它们内部是怎么工作的

TensorFlow-import-data

Posted on 2017-10-06 | Edited on 2019-05-03 | In Tensorflow

Importing Data

利用tf.data API 接口可以很方便的建立复杂的输入管道，不论是图像数据、文本数据还是序列数据，利用tf.data都可以很简单的处理大量的数据，不同类型的数据，以及完成复杂的数据转换。

tf.data引入了两种抽象的方法：

  • tf.data.Dataset 这个代表了一系列的包含很多张量的元素，比如对图像管道，一个元素就是一个训练样本，代表图像数据的张量和一组标签。创建一个dataset的方法主要有两种：

    • Dataset.from_tensor_slices() 从一个或多个张量对象中创建dataset
    • Dataset.batch() 应用转换的方法。从Dataset对象中创建
  • tf.data.Iterator() 提供了从dataset中提取元素的主要方法。当Dataset执行的时候，通过调用Iterator.get_next(),可以得到下一个元素。

本节讲述的是怎样创建各种不同的dataset和iterator并从中提取数据。
