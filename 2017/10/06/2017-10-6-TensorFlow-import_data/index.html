Importing Data

利用tf.data API 接口可以很方便的建立复杂的输入管道，不论是图像数据、文本数据还是序列数据，利用tf.data都可以很简单的处理大量的数据，不同类型的数据，以及完成复杂的数据转换。

tf.data引入了两种抽象的方法：

  • tf.data.Dataset 这个代表了一系列的包含很多张量的元素，比如对图像管道，一个元素就是一个训练样本，代表图像数据的张量和一组标签。创建一个dataset的方法主要有两种：

    • Dataset.from_tensor_slices() 从一个或多个张量对象中创建dataset
    • Dataset.batch() 应用转换的方法。从Dataset对象中创建
  • tf.data.Iterator() 提供了从dataset中提取元素的主要方法。当Dataset执行的时候，通过调用Iterator.get_next(),可以得到下一个元素。

本节讲述的是怎样创建各种不同的dataset和iterator并从中提取数据。

Import data

如果你的数据可以一次性的导入到内存之中，可以利用

  • tf.data.Dateset.from_tensor()
  • tf.data.Dataset.from_tendor_slices()

来导入数据。否则可以使用

  • tf.data.Dataset.TFRecordDataset()

来导入数据.

dataset中包含有具有同种结构的元素，每个元素包含一个或多个张量，即tf.Tensor对像。每个元素具有两种属性：

  • tf.Dtype
  • tf.Tensorshape

可以通过对应的方法来访问两种属性：

  • Dataset.output_types
  • Dataset.output_shapes

Creating an iterator

使用Dataset中数据最常用的方法是创建一个Iterator对象，它提供了一次访问dataset中一个数据的方法。一个Iterator对象提供了两个操作：

  • Iterator.Intializer() 用来初始化迭代状态
  • Iterator.get_next() 返回dataset中下一元素的tf.Tensor对象。

一旦创建了代表输入数据的Dataset对象，下一步就是创建一个Iterator去访问Dataset中的每个元素。tf.data一共支持4中类型的Iterator:

  • one-shot 一次遍历整个dataset, 不需要显示的初始化。可以处理几乎所有基于队列输入的管道数据，但是不支持参数化的输入。

  • initializable 需要显示的初始化操作 iterator.initializer(), 允许参数化定义dataset, 即使用tf.placeholder()。

  • reinitializable 可以初始化多个不同的dataset对象。比如讲数据集划分为训练集个验=验证集，他们是不同的Dataset对象，但是有想用的结构。reinitializable iterator是通过结构来定义的，亦即tf.data.Iterator.from_structure()

  • feedable 可以和tf.placeholder()一起用于选择Iterator用于tf.Session.run(), 通过配置feed_dict参数。它提供了和reinitialzable类似的功能， 但是不需要开始切换iterator的时候初始化。
