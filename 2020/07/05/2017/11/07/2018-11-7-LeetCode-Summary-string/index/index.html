LeetCode 刷题总结（一）

这是字符跳动推荐题目的总结。

字符串类

作为一名优秀的工程师，熟练使用和操作字符串是必须的技能。以下是遇到的一些题目的总结与反思。

无重复字符的最长子串

给定一个字符串，请你找出其中不含有重复字符的 最长子串 的长度。

示例 1:

输入: “abcabcbb”
输出: 3
解释: 因为无重复字符的最长子串是 “abc”，所以其长度为 3。
示例 2:

输入: “bbbbb”
输出: 1
解释: 因为无重复字符的最长子串是 “b”，所以其长度为 1。
示例 3:

输入: “pwwkew”
输出: 3
解释: 因为无重复字符的最长子串是 “wke”，所以其长度为 3。
请注意，你的答案必须是 子串 的长度，”pwke” 是一个子序列，不是子串。

此题的线性时间内的解法为：首先维护一个子串数组，开始循环前将第一个数据送入子串数组中。在每个循环体中遍历子数组是否有与当前值相等的情况。若没有，则将当前值加入子数组；若有则删除子数组中包括重复位置及之前的所有值，再将当前值添加到子数组中。最后子数组的长度就是子串长度

字符串的排列

给定两个字符串 s1 和 s2，写一个函数来判断 s2 是否包含 s1 的排列。

换句话说，第一个字符串的排列之一是第二个字符串的子串。

示例1:

输入: s1 = “ab” s2 = “eidbaooo”
输出: True
解释: s2 包含 s1 的排列之一 (“ba”).

示例2:

输入: s1= “ab” s2 = “eidboaoo”
输出: False

注意：

输入的字符串只包含小写字母
两个字符串的长度都在 [1, 10,000] 之间

如果考虑暴力枚举子串s1的所有排列情况，时间复杂度是指数级的，无法有效求解，需要再做思考。如何衡量两个子串同属一个排列？稍加思考后我们发现，可以考虑使用子串中每个字符出现的频率，每个字符频率都相同的则属于同一个字符串。

字符串相乘

字符串乘法属于大数乘法问题。当两个数超过了计算机可以存储的最大位数，或者相乘后的结果超出了数值类型可以保存的最大位数，即需要保存为字符串型的变量。大数乘法主要有以下几种方法：

  • 模拟竖式乘法，逐位相乘再累加。
  • 分治乘法
  • FFT
  • 中国剩余定理：把每个数分解到一些互素的模上，然后每个同余方程对应相乘。

这里先介绍下第一种方法：模拟竖式乘法。实现这种方法有两个下技巧：

  • 将两个字符串逆序存储，方便之后从左向右计算进位。
  • 先不计算进位，将第i位和第j位的计算结果保存在结果数组的第i+j位上

两个小技巧可以大大简化代码复杂度。
